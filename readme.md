# House Price Analysis
Interactive Data Analysis Tool for House Price Dataset

## Project Description
This is a simple interactive data analysis application built with Streamlit. It provides data overview, missing value analysis, sale price distribution, correlation analysis, and visualization of key features related to house prices.

## Files
- app.py: Main application code
- train.csv: Dataset used for analysis
- requirements.txt: Required Python libraries
- house_price_analysis.ipynb: Jupyter Notebook for data analysis

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

## Data Source
Kaggle House Prices: Advanced Regression Techniques
